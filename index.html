Program Hello World

Menampilkan Teks "Hello World" ke console:

#include <iostream>
using namespace std;
int main() {
   cout << "Hello World";
   return 0;
}

Teks dibungkus dalam " " yang menandakan bahwa tipe data yang ingin kita gunakan dan tampilkan berupa tipe data string.


Membuat Variabel

C++ memiliki berbagai macam tipe data -sama seperti bahasa program lainnya-, diantaranya yang paling sering kita temui adalah tipe data int, float, char dan string serta boolean -dan masih banyak lainnya-.

Dengan memahami tipe data dapat mempermudah pemrograman C++ -dan bahasa program lainnya-.
Berikut penjelasan singkat beberapa tipe data yang telah diuraikan sebelumnya:


int
tipe data yang menyimpan angka, -dapat menampung angka negatif-.


float
tipe data yang menyimpan angka, perbedaannya terletak pada penggunaan koma pada angka.


char
char merupakan kependekan dari character. Tipe data ini digunakan dalam penggunaan karakter -sederhananya, abjad, angka, simbol-simbol (yang ada dalam sistem komputer)-.
Contohnya:
A a @, dan lain-lain.


boolean
tipe data ini hanya memiliki dua nilai: yakni true&false saja. Konsepnya sama seperti bilangan biner,
nilai 1 untuk nilai true dan
nilai 0 untuk nilai false

*Untuk penggunaannya di dalam program C++, harus dengan menggunakan keyword bool.


string
tipe data ini sebenarnya tipe data gabungan. Terbentuk dari kumpulan karakter-karakter (char) yang membentuk sebuah teks/kalimat.

Tata cara penamaan variabel:

  1. Variabel diawali dengan tipe data yang ingin digunakan
    Contoh:

    int variabelSaya = 120;

  2. Variabel tidak boleh diawali dengan angka. Sebaiknya variabel diawali dengan huruf (huruf kecil/besar). Dan jangan mengawali penamaan variabel dengan menggunakan simbol-simbol.

  3. Setelah mengawali nama variabel dengan huruf, maka setelah itu ke depannya dapat diberikan angka-angka atau simbol -tidak mendukung semua simbol-.

    int Var120_ = 120;

  4. Nilai dari variabel dapat mengandung variabel lainnya, contoh:

    int variabelSayaKedua = variabelSaya;

  5. Dan lain-lain. (untuk lebih lengkapnya, dapat dilihat di perangkat dengan menggunakan web browser dan internet)

#include <iostream>
using namespace std;
int main() {
   // variabel int
   int TipeDataInteger = 4;
   // variabel float
   float TipeDataFloat = 2.0;
   // variabel char
   char TipeDataChar = 'A';
   // *variabel dengan tipe data char, nilainya perlu dibalut oleh tanda kutip satu '' 
   // bukan tanda kutip dua "" 
   // variabel boolean
   bool TipeDataBoolean = true;
   // variabel string
   string TipeDataString = "Hello World!";
   // *variabel dengan tipe data string, nilainya perlu dibalut oleh tanda kutip dua ""
   // bukan tanda kutip satu ''
   return 0;
}

tipe data int dan float dapat dioperasikan dengan menggunakan operator aritmatika. Dapat ditambah, dikurang, dikali, dibagi dan sisa bagi (modulus).


Menampilkan Variabel

#include <iostream>
using namespace std;
int main() {
   // variabel int
   int TipeDataInteger = 4;
   // menampilkan tipe data integer
   cout << TipeDataInteger << endl;
   // menampilkan tipe data integer dengan menggunakan operasi aritmatika
   cout << TipeDataInteger + 6 << endl;
   // variabel float
   float TipeDataFloat = 4.0;
   // menampilkan tipe data float dengan menggunakan operasi aritmatika
   cout << TipeDataFloat / 2 << endl;
   // variabel char
   char TipeDataChar = 'A';
   // menampilkan tipe data char
   cout << TipeDataChar << endl; 
   // variabel boolean
   bool TipeDataBoolean = true;
   // menampilkan tipe data boolean
   cout << TipeDataBoolean << endl;
   // variabel string
   string TipeDataString = "Hello World!";
   // menampilkan tipe data string
   cout << TipeDataString << endl;
   return 0;
}
